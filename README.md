# ICS Tetris

A tetris clone.

## Controls
| Key   | Action                   |
| ----- | ------------------------ |
| Up    | Rotate clockwise         |
| Z     | Rotate counter-clockwise |
| A     | Rotate 180 degrees       |
| LEFT  | Move left                |
| RIGHT | Move right               |
| DOWN  | Move down (softdrop)     |
| SPACE | Harddrop                 |
