# ICS Tetris

A tetris clone.

## Controls
| Key   | Action                   |
| ----- | ------------------------ |
| Up    | Rotate clockwise         |
| Z     | Rotate counter-clockwise |
| A     | Rotate 180 degrees       |
| LEFT  | Move left                |
| RIGHT | Move right               |
| DOWN  | Move down (soft drop)    |
| SPACE | Hard drop                |

## Scoring

Scoring is done as suggested in the 2009 Tetris Design Guideline, with the exception that Mini T-Spins are not implemented.
